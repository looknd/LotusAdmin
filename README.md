##LotusAdmin是什么?
一个使用thinkphp5.0.11+layui2.0后台开发的极速开发框架

##向thinkphp致敬
* 基于layuui2.0+thinkphp5.0

##LotusAdmin有哪些功能？
* 极简--极简和响应式的高颜值响应式布局，谁说后台都很丑
* 安全--每一个字段都有严格的前后端验证
* 权限--自带后台用户权限
    *  用户管理
    *  节点管理
    *  角色授权
* 导航--前台导航管理
